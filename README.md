# EyeCare Ecommerce Microservices

## Overview

Enterprise EyeCare Ecommerce Platform built using Java, Spring Boot, Spring Cloud, Kafka, Docker, Kubernetes and AWS.

## Technology Stack

- Java 21
- Spring Boot
- Spring Cloud
- Spring Security
- JWT
- Spring Data JPA
- Hibernate
- MySQL
- Kafka
- OpenFeign
- Docker
- Kubernetes
- AWS
